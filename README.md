# bat

自分用に作ったバッチファイル  
汎用性があまりないです  

## 一覧
- 7zipa
  - 引数にあるディレクトリを7zipで圧縮
- backupdir
  - ディレクトリをアーカイブ化
- backupworld
  - Minecraftのワールドデータをバックアップ
- jpgtopng
  - .jpgを.pngに変換
- switch_jpgtopng
  - SDカード内のSwitchのアルバムをpngに変換しローカルにコピー
- ugoiratomp4
  - zip圧縮されたPixivのうごイラをpngとmp4に変換
- waifu2x_x64
  - tanakamura版waifu2x-converter( https://github.com/tanakamura/waifu2x-converter-cpp )を実行
